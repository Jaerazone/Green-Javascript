※※※※※일치 연산자※※※※※※※※※※※
=== 자료형과 그 값이 같을때 참
!== 자료형과 그 값이 다를때 참(둘다 다르거나, 하나만 달라도 참)

0==false
true	// 0은 false라서 참

0===false
false	// ===는 자료형으로 체크되어서
	   0은 숫자, false는 문자라 거짓

0 !== false
true	// !== 는 좌우 비교했을때 값이다르면 참!
